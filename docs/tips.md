# Tips & Notes

- When cloning **Chinese dialects**, provide both `ref_audio` and `instruct` (e.g., `ref_audio="sichuan.wav", instruct="四川话"`) for more stable dialect output.
- **Hokkien (闽南话)** can only be synthesized using [Tai-lo romanization](https://en.wikipedia.org/wiki/T%C3%A2i-l%C3%B4) as input; Chinese characters are not supported for Hokkien in the current model version.
